# Projeto Azure Databricks

Projeto desenvolvido para demonstrar o uso do Azure Databricks com notebooks versionados no GitHub.

## Tecnologias utilizadas
- Azure Databricks
- Apache Spark
- Python
- GitHub

## Funcionalidades
- Criação de cluster no Databricks
- Processamento de dados com Python
- Análise de dados
- Visualização de informações

## Estrutura do projeto
- Notebook de análise
- Arquivo CSV de dados
- Documentação
